Capa Asignada para el Análisis de Rugosidad

En este caso, se realizó la capa de Población considerando la Cantidad de Habitantes del estado de Colima.

  1. Documento de Exposición de Procedimiento

    2. Primero se hizo el intento de obtener una capa que diera prioridad a las localidades de mayor población, sin embargo, solo se logró hacer una que dijera qué tan lejos se está de una locación.

    Diapositiva procedimiento (Capas mal logradas)

    Al final, después de un análisis con ayuda del profesor, se logró crear interpolaciones con respecto al número de habitantes de cada localidad, dándonos como resultado las capas esperadas.

    Esta capa se logró utilizando las capas "localidad250_a" (área de asentamientos grandes) y "localidad250_p" (puntos de asentamientos rurales) del estado de Colima.

    Se usaron las herramientas IDW (Distancia Inversa Ponderada) para la capa "localidad250_p" e Interpolación Kernel con Barreras para "localidad250_a" para finalizar con una Suma Ponderada de estos dos ráster.

  2. Proceso, Datos y Resultados por Herramienta

    3. Todos los procesos se realizaron con herramientas del software ArcGIS en este orden:

    1. Suma Ponderada (Evaluación Multicriterio)

      2. Se deben importar todas las capas ráster necesarias para el análisis de rugosidad (7 capas) y darles diferentes pesos dependiendo cuál se concidere de mayor importancia.

    2. Acumulación de Distancia

      3. Requiere de 2 capas, Orígenes de ráster o entidad de entrada (punto Origen) y el Ráster de costes de entrada (Suma ponderada), para dar como resultado 2 ráster, el Ráster de acumulación de distancia de salida y el Ráster de dirección hacia atrás de salida.

    3. Ruta Óptima como línea

      4. Requiere de 3 capas, Destinos de ráster o entidad de entrada (punto Destino), Ráster de acumulación de distancia de entrada y el Ráster de dirección de flujo o dirección hacia atrás de entrada, para dar 1 Ráster de salida de Ruta óptima de salida como línea.

  3. Proceso Evaluación Multicriterio

    4. Para realizar la evaluación multicriterio fue necesario dividir las 7 capas (más 2 capas de otros estados) entre todos los compañeros del grupo, por lo que cada quién realizó su parte por separado y después los compartimos para realizar este ejercicio.

    La manera en como lo dividimos fue la siguiente:

    • Jorge: Red hidrográfica.
    • Natalia: Usos de suelo y vegetación.
    • Sebastian: Áreas Naturales Protegidas.
    • Isaac: Población.
    • Oscar: Cuerpos de agua.
    • Omar: Línea de transmisión eléctrica.
    • Diana: Vías Férreas.
    • Karla: Susceptibilidad a movimiento de masas (Capa de otro estado)
    • Tapia: Susceptibilidad a subsidencia. (Capa de otro estado)

    Si se desea realizar cambios para encontrar una ruta diferente, se pueden cambiar los pesos de las capas, por ejemplo, si se quiere evitar a toda costa un río o un lago, las capa de Red Hidrográfica y Cuerpos de agua deben tener conciderablemente un peso mayor que las otras.
